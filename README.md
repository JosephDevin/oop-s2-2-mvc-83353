# Food Safety Inspection Tracker

An ASP.NET Core MVC application for tracking food premises inspections, outcomes, and follow-up actions.

This helps a local council tracks food premises inspections. Inspectors record inspection
outcomes and follow-ups. Management wants visibility into what’s happening and
when things fail.
